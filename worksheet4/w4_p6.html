1. What is the difference between Phong shading and Phong lighting (the Phong reflection model)?

    2. The Phong lighting calculates the amount of light that is reflected towards the viewer. It is an ilumination model that does this with the components: ambient, diffuse and specular.

    Meanwhile, the Phong shading is a shading model that uses an ilumination model to get and return the colour for each pixel.


  2. What is the difference between flat shading, Gouraud shading, and Phong shading? List pros and cons of each. Is Gouraud or Phong shading the best method for simulating highlights? Explain.

    3. Flat shading: each polygon has only one normal vector, therefore all pixels in the same polygon have the same ilumination value. Pros: fastest and simplest shading model. Cons: the results are meshes with edges that are very easy to spot.

    Gouraud shading: each polygon has one normal vector in each vertex where the ilumination value is calculated and then all the values from each vertex of the polygon are interpolated for all the pixels in the polygon. Pros: faster than Phong shading and better quality than Flat shading. Cons: worse quality than Phong shading.

    Phong shading: each polygon has one normal vector in each vertex. Then for all pixels in the polygon the normals are interpolated. Lastly, for each pixel the ilumination value is calculated using its normal. Pros: the best quality. Cons: takes the longest to calculate.

    For simulating highlights Phong shading is better as it gets smoother results of the surfaces.


  3. What is the difference between a directional light and a point light?

    4. Directional light does not have a position, making the rays parallel. While the rays of a point light radiate outward from the position of the point light.


  4. Does the eye position influence the shading of an object in any way?

    5. It influences in the way the specular light is calculated, therefore it influences the shading.


  5. What is the effect of setting the specular term to (0,0,0)?

    6. That there is no specular lighting.
